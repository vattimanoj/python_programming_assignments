# Python Programming Assignments

A collection of Jupyter notebooks containing beginner-to-intermediate Python practice problems and solutions, covering core data structures and basic programming exercises. Each notebook poses a series of questions with sample input/output, followed by a worked solution in a code cell.

## Contents

| Notebook | Topic | Description |
|---|---|---|
| `Python List.ipynb` | Lists | Problems on counting elements, finding max/min, removing items, computing averages, checking emptiness, and other list manipulations — all implemented without relying on built-in shortcuts like `len()` where specified. |
| `Python Strings .ipynb` | Strings | Problems on counting characters, reversing strings, palindrome checks, finding the most common character, and checking anagrams — done manually instead of using slicing tricks like `[::-1]`. |
| `Python_Dictionary .ipynb` | Dictionaries | Problems on key/value lookups, updating dictionaries, checking key existence, summing values, listing keys/values, counting entries, and merging dictionaries. |
| `Tuesday_work.ipynb` | Mixed exercises | A set of general practice problems involving conditionals, loops, string slicing, and list iteration (e.g. conditional sum/product, paired iteration over two lists, removing empty strings). |

## Format

Each question in a notebook follows the same structure:

1. **Question** — a markdown cell describing the task.
2. **Sample input/output** — example values and the expected result.
3. **Solution** — a code cell with a working Python implementation.

## Requirements

- Python 3
- Jupyter Notebook or JupyterLab (or any compatible environment, such as VS Code or Google Colab)

## Usage

Clone the repository and open any notebook with Jupyter:

```bash
git clone https://github.com/vattimanoj/python_programming_assignments.git
cd python_programming_assignments
jupyter notebook
```

Then open the notebook of interest and run the cells in order.
